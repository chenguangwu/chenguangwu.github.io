git 版本回退整理：


1.本地回退

[sql] view plain copy
  1. //按文件、路径回退==仅回退这个目录或者文件  
  2. git reset HEAD path/  
  3.   
  4. //按版本回退==回退到上个版本  
  5. git reset HEAD^  
  6.   
  7. //按版本号回退==回退到某个版本  
  8. git reset –-soft HEAD~3    
  9. git reset 057d    
  10.   
  11. //将本地的状态回退到和远程的一样    
  12. git reset –-hard origin/master    

2.远程强制回退

//强制回退远程提交
git push origin HEAD --force
