用javascript判断一个html元素是否存在的五种方法：
1. 判断表单元素是否存在（一）
if("periodPerMonth" in document.theForm){
return true;
}else{
return false;
}
2. 判断页面元素是否存在
if(document.getElementById("XXX")){
//存在
}
3. 判断表单元素是否存在（二）
if(document.theForm.periodPerMonth){
//存在
}
或
if(typeof(document.theForm.periodPerMonth)=="object"){
//存在
}
4. 判断表单是否存在
if(document.theForm){
//存在
}
5.用Jquery写脚本
if ( $("#someID").length > 0 ){
$("#someID").text("hi");
}
