一行命令搞定node.js 版本升级

投稿：hebedich 字体：[增加 减小] 类型：转载 时间：2014-07-20 我要评论

今天，又发现一个超级简单的升级node.js的方法。一行命令搞定，省去了重新编译安装的过程。
. .

node有一个模块叫n（这名字可够短的。。。），是专门用来管理node.js的版本的。

首先安装n模块：

?
1
npm install -g n

第二步：

升级node.js到最新稳定版

?
1
n stable

是不是很简单？！

n后面也可以跟随版本号比如：

?
1
n v0.10.26

或

?
1
n 0.10.26

就这么简单，这可怎么办？？！！
