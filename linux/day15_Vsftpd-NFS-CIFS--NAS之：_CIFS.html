NAS之： CIFS



NAS之CIFS
CIFS： Common Internet File System Windows和Unix系统之间共享文件的一种协议
========================================================

一、Samba服务器 (存储端)
借助于Linux平台上的Samba软件来实现CIFS共享
软件包: samba,samba-common
配置文件: /etc/samba/smb.conf
监听端口: 445/tcp 137,138,139/tcp
服务名 ：smb

案例1：
1. 安装软件
[root@yangs ~]# rpm -qa |grep samba
samba-common-3.0.33-3.37.el6
samba-3.0.33-3.37.el6

2. 建立共享所需要目录
[root@yangs ~]# mkdir -p /share/{smb1,smb2}
[root@yangs ~]# chmod 777 /share/smb1 /share/smb2
[root@yangs ~]# touch /share/smb1/file1
[root@yangs ~]# touch /share/smb2/file2

3. 通过CIFS共享
[root@yangs ~]# vim /etc/samba/smb.conf
#======================= Global Settings ================
[global]
security = user|share|domain|ads|server //定义安全级别

#======================= Share Definitions ===============
[smb1] 共享名和文件夹名可以不一样
path = /share/smb1

[smb2]
path = /share/smb2
comment = music
valid users = alice jack robin
writable = yes

4. 建立Samba用户
[root@yangs ~]# useradd alice //建立系统用户
[root@yangs ~]# smbpasswd -a alice //设置samba密码 有自己专门的用户名密码

5. 启动Samba服务
[root@yangs ~]# service smb restart
[root@yangs ~]# chkconfig smb on


二、客户端连接测试
Windows端
1. \\192.168.2.168
2. 映射网络驱动器

# yum -y install samba-client

三、Samba扩展配置
1. 取消用户HOME共享
[root@yangs ~]# vim /etc/samba/smb.conf
;[homes]
; comment = Home Directories
; browseable = no
; writable = yes
; valid users = %S
; valid users = MYDOMAIN\%S

2. 存储端实现访问控制
基于用户的访问控制 user-based
基于主机的访问控制 host-based

案例1：samba实现公司文件服务器 (也可看成是为多个Project准备NAS存储)
a. 建立所需的组和用户
[root@yangs ~]# groupadd hr
[root@yangs ~]# groupadd it
[root@yangs ~]# useradd -G hr hr01
[root@yangs ~]# useradd -G hr hr02
[root@yangs ~]# useradd -G it it01
[root@yangs ~]# useradd -G it it02
[root@yangs ~]# smbpasswd -a hr01
[root@yangs ~]# smbpasswd -a hr02
[root@yangs ~]# smbpasswd -a it01
[root@yangs ~]# smbpasswd -a it02

b. 目录、权限
[root@yangs ~]# mkdir /home/hr
[root@yangs ~]# mkdir /home/it
[root@yangs ~]# mkdir /home/public
[root@yangs ~]# chmod 1770 /home/hr //3770: set gid, sticky
[root@yangs ~]# chmod 1770 /home/it
[root@yangs ~]# chgrp hr /home/hr
[root@yangs ~]# chgrp it /home/it
[root@yangs ~]# ll -d /home/hr /home/it
drwxrwx--T 2 root hr 4096 11-12 10:45 /home/hr
drwxrwx--T 2 root it 4096 11-12 10:46 /home/it

c. samba共享
基于用户的访问控制：
[root@yangs ~]# vim /etc/samba/smb.conf
[public]
path = /home/public
guest ok = yes
[hr]
path = /home/hr
valid users = +hr
writable = yes
[it]
path = /home/it
valid users = @it
write list = it01
[root@yangs ~]# service smb restart


基于主机的访问控制：
==针对整个服务器
[global]
hosts allow = 192.168.2. 192.168.3.

==针对某个共享目录
[hr]
path = /home/hr
valid users = +hr
writable = yes
create mask = 0775 //上传的文件默认权限为0775
hosts allow = 192.168.3.
[it]
path = /home/it
valid users = @it
write list = it01
hosts allow = 192.168.2.
